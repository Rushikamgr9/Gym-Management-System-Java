**PROJECT TITLE:**
Gym Management System Java 

**PURPOSE OF PROJECT:**
The purpose of this project is to develop a Java-based Gym Management System that helps manage gym members efficiently using a graphical user interface (GUI). The system allows staff to handle both regular and premium members, track membership status, manage payments, and maintain records. It reduces manual work, minimizes errors, and improves overall efficiency in gym operations.

**HOW TO START THIS PROJECT:**

1. Open the project folder in Visual Studio Code or any Java IDE.
2. Compile the Java files using:
   javac GymGUI.java
3. Run the program using:
   java GymGUI
4. The GUI window will open, and you can start using the system.

**USER INSTRUCTIONS:**

* Enter member details in the provided text fields.
* Use “Add Regular Member” or “Add Premium Member” to add members.
* Use “Activate” or “Deactivate” to manage membership status.
* Use “Upgrade Plan” to convert regular members to premium.
* Use “Calculate Discount” for applicable members.
* Use “Pay Due Amount” to process payments.
* Use “Remove Member” to delete a member.
* Use “Display” to view all member records.
* Use “Clear” to reset input fields.
* Use “Save” and “Read” to store and retrieve data from file.
